## Development

For a working demo, clone this repo and run:

1. Install all dependencies

    ```shell
    npm install
    ```

    or

    ```shell
    yarn
    ```

2. Start

    ```shell
    npm start
    ```

    or

    ```shell
    yarn start
    ```

3. Open `http://localhost:4200` in your browser.
